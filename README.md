# Pokemon App

## Technologies stack

- TypeScript
- React
- RTK Query
- Styled components
- Eslint
- MUI

## Instructions

1. Clone repo[https://github.com/GrynenkoTimur/pokemon-app] to your local machine.
2. Run `npm install`
3. Create `.env` file with string `REACT_APP_POKEMON_API_URL="https://pokeapi.co/api/v2"`
4. Run `npm start`
